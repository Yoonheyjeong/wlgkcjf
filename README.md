### 지하철

## 프로젝트 소개
이 프로젝트는 시각장애인 이동약자가 지하철에서 문을 찾을때 겪는 이동 장벽을 3D로 체험할 수 있도록 만든 웹 시뮬레이터입니다.

## 이동약자 유형
- 시각장애인 

## 주요 장면
-지하철 문까지 가는 과정

## 조작 방법
- W: 앞으로 이동
- A: 왼쪽 이동
- S: 뒤로 이동
- D: 오른쪽 이동
- 마우스: 시점 이동
